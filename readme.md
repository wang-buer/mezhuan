# 基于springboot的美妆购物管理系统2



## 1、项目介绍

基于springboot的美妆购物管理系统拥有三种角色

用户：登录注册、收藏评论、购物车、我的订单、添加收货地址等

商家：发货

管理员：用户商家管理、商品管理、类型管理、订单管理、公告管理等

## 2、项目技术

后端框架：springboot、Mybatis

前端技术：前端html、后端VUE前后分离

## 3、开发环境

- JAVA版本：JDK1.8（最佳）
- IDE类型：IDEA、Eclipse都可运行
- 数据库类型：MySql（5.7、8.x版本都可） 
- tomcat版本：无需
- maven版本：无限制
- 硬件环境：Windows
